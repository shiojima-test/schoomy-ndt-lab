# SchooMy NDT Lab v4.0

スクーミーオレンジボード × Web Serial API × フーリエ変換 × 虚数 を体験する教材ダッシュボードです。

## できること

- Web Serial API でオレンジボードと直接接続
- シリアル通信で送られてくるアナログ値（0-1023）をリアルタイム表示
- 任意のセンサーに対応（音/光/振動/加速度/磁力など）
- 「フーリエ変換ON」ボタンで周波数スペクトラムを表示
- リアルタイム統計（ピーク周波数、RMS、最大・最小・平均）
- 計測スタート/ストップで連続記録、CSV保存
- マイクなし環境でもデモできるシミュレーションモード搭載

## 使い方

1. ブラウザで [公開URL](https://shiojima-test.github.io/schoomy-ndt-lab/) を開く
2. オレンジボードをUSB接続
3. センサー種別と計測対象を入力
4. 「🔌 接続」→ ポート選択
5. 「⏺ 計測スタート」→ 物を叩く・揺らす
6. 「📊 フーリエ変換 ON」で周波数表示
7. 「⏹ 計測ストップ」→ CSVダウンロード

## オレンジボード側のサンプルプログラム

```cpp
// アナログピンA0のセンサー値を10msごとに送信
void setup() {
  Serial.begin(9600);
}
void loop() {
  int v = analogRead(A0);
  Serial.println(v);
  delay(10);
}
```

## ブラウザ要件

Web Serial API対応のブラウザが必要：
- Chrome / Edge / Opera（最新版）
- Safari, Firefox は非対応

## 教育的なねらい

- センサーが取るのは「ただの時間変化する数値」
- フーリエ変換でその数値を周波数成分に分解できる
- 数学的基盤は虚数（複素数）
- 音センサーでも光センサーでも、同じ仕組みで解析できる
- 橋・トンネルの打音検査、新幹線の車輪点検、MRIなど社会の様々な場面で使われている技術

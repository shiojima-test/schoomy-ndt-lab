# SchooMy NDT Lab v5.0

スクーミーオレンジボード × Web Serial API × フーリエ変換 × スナップショット比較

## v5.0 の特徴：叩いて比較する

- **📸 今を分析** ボタンで、その瞬間の周波数パターンを保存
- 最大 **4枚まで** 並べて比較できる
- 「正常な陶器」と「ヒビの陶器」のピーク周波数の違いがひと目で分かる

## できること

- Web Serial API でオレンジボードと直接接続
- シリアル通信で送られてくるアナログ値（0-1023）をリアルタイム表示
- 任意のセンサーに対応（音/光/振動/加速度/磁力など）
- **Y軸レンジの調整**（AUTO / 手動）でセンサーの変動を見やすく
- **FFTスナップショット比較**（最大4枚）
- リアルタイム統計（RMS、最大・最小・平均）
- 計測スタート/ストップで連続記録、CSV保存
- シミュレーションモード搭載（マイクなし環境でもデモ可）

## 使い方

1. オレンジボードをUSB接続
2. センサー種別と計測対象を入力
3. 「🔌 接続」→ ポート選択
4. 物にセンサーを当てて叩く・揺らす
5. 「📸 今を分析」を押す → 1枚目のスナップショットが保存される
6. 別の対象で同じ手順 → 2枚目に保存
7. 4枚並べて、ピーク周波数を比較

## オレンジボード側のサンプルプログラム

```cpp
void setup() {
  Serial.begin(9600);
}
void loop() {
  int v = analogRead(A0);
  Serial.println(v);
  delay(10);
}
```

## ブラウザ要件

- Chrome / Edge / Opera（最新版）
- Safari, Firefox は Web Serial API 非対応

## 教育的なねらい

- センサーの値は「ただの時間変化する数値」
- フーリエ変換で「**叩いた音の指紋**」が見える
- 正常な物と異常な物では「指紋」が違う
- これが橋・トンネルの打音検査、新幹線の車輪点検の原理

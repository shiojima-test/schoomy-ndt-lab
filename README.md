# SchooMy NDT Lab

非破壊検査（NDT: Non-Destructive Testing）× フーリエ変換 × 虚数 を体験するための教材ダッシュボードです。

## できること

- 音センサー（S-CN-A06）の値をリアルタイム表示
- マイクで叩いた音を取得し、時間波形を表示
- フーリエ変換（FFT）で周波数スペクトラムを表示
- リアルタイム統計（ピーク周波数、RMS、最大・最小・平均振幅）
- 計測スタート/ストップで連続記録
- 波形CSV / FFT CSV の2形式でダウンロード
- シミュレーションモード（マイクなし環境でもデモ可能）

## 使い方

1. ブラウザで [公開URL](https://shiojima-test.github.io/schoomy-ndt-lab/) を開く
2. 計測対象名を入力
3. 「計測スタート」を押す（マイク許可）
4. 物にセンサーを当てて叩く or シミュレーションボタン
5. 「計測ストップ」→ CSVダウンロード

## 教育的なねらい

- センサーが取るのは「ただの時間変化する数値」
- フーリエ変換でその数値を周波数成分に分解できる
- 分解の数学的基盤は虚数（複素数）
- 同じ仕組みは橋・トンネルの打音検査、新幹線の車輪点検、医療のMRIなどに使われている
